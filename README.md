# BoardGames
Android Studio project created for the "Ubiquitous Systems" course at Poznan University of Technology.
The app uses kotlin and BoardGameGeek API to fetch users collections and store them localy.
Features include:
- Taking pictures with camerax api
- MySql database
- Game list using RecyclerView for efficiency
- Picking photos from the phone galery and assigning them to games
- loading pictures with the Coil library
